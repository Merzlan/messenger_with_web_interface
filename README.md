Проект собирается и запускается следующими командами из директории проекта: Mkdir build – создаём папку, куда собираем Cd build – заходим в нее Cmake .. читаем cmakelists.txt и генерируем makefile Make -j4 – читаем сгенерированный makefile и запускаем компилятор, флаг -j4 означает использовать 4 параллельных потока – компиляция идёт быстрее ./server – запускаем скомпилированный бинарник
